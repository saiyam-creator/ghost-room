// THE APP

Ghost Room is an anonymous, ephemeral real-time chat application. It was built with one goal — let people communicate freely without leaving any digital footprint.

There are no accounts, no servers storing your data permanently, and no way to trace conversations back to you. Create a room, share the key, chat — and when you're done, destroy it.

Powered by Supabase for real-time database and file storage. Zero backend, zero framework, zero complexity.
